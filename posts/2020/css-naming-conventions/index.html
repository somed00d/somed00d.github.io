Thinking about guidelines and conventions

Since this website is pretty basic and supposed to be lean I decided that I wouldn't use a front-end framework like I did with narco.news. This means that I'll have to write all of the styles myself. At the moment what I've written style wise is the bare minimum in order to provide something. But as this website continues to grow I'll be required to expand on that.
